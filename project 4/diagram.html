🌤️ Weather Forecast — XML Structure

How the data is organised in weather.xml

<weatherdata>
<city name="Lagos"> × 7 name attr
<temperature>
unit="°C"
32
<humidity>
unit="%"
78
<windspeed>
unit="km/h"
14
<condition>
Sunny

🔑 Legend

Root element — wraps all city data
Repeating element — one per city (×7), has a name attribute
Child element — one weather reading, some have a unit attribute
Example value stored inside the element
XML attribute (written inside the opening tag)
